A modern, responsive web application designed to simulate investment growth over time. This tool allows users to calculate compound interest, estimate future returns with monthly contributions, and visualize financial data through interactive charts.

## ‚ú® Key Features

- **ROI Simulation:** Calculate future investment value using compound interest formulas.
- **Customizable Parameters:** Adjust principal amount, annual interest rate, duration, and compounding frequency.
- **Recurring Contributions:** Support for regular monthly or annual deposits.
- **Interactive Charts:** Data visualization using Recharts
- **Responsive Design:** Optimized for mobile and desktop devices.
